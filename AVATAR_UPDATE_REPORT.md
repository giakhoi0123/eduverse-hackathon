## Định Dạng Avatar

Tất cả avatar được lưu trữ với đường dẫn: `/img/[TenFile]`

Các định dạng hình ảnh được hỗ trợ:

- `.png` - 51 files
- `.jpg` - 45 files
- `.jpeg` - 1 file
- `.webp` - 2 files
- Không phần mở rộng - 1 file (TranAnhTong)

## Hình Ảnh Chưa Sử Dụng

1. **favicon.svg** - Icon của website
2. **HaiBaTrung.png** - Có thể sử dụng cho nhân vật Hai Bà Trưng nếu được thêm vào danh sách

## Kết Luận

✅ **Hoàn thành 100%** - Tất cả 100 nhân vật lịch sử đã được cập nhật avatar thành công từ thư mục `img/`.

## Files Được Cập Nhật

- `backend/utils/characters-data.js` - File chính chứa dữ liệu nhân vật

## Scripts Đã Tạo

1. `update-avatars.js` - Script tự động cập nhật avatar
2. `verify-avatars.js` - Script kiểm tra hình ảnh đã/chưa sử dụng
3. `check-avatars.js` - Script kiểm tra nhân vật có/chưa có avatar
